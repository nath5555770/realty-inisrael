⚙ Activation du CMS — 1 étape requise

Les nouvelles fonctionnalités Textes et Réglages nécessitent deux tables Supabase qui n'ont pas encore été créées sur ton projet.

  1. Ouvre l'éditeur SQL de Supabase : supabase.com/dashboard/project/_/sql
  2. Vérifie que tu es bien connectée au projet realty-inisrael
  3. Copie le contenu du fichier de migration ci-dessous puis colle-le dans l'éditeur SQL
  4. Clique sur "Run" en bas à droite
  5. Reviens ici et clique sur "J'ai exécuté le script"
